# addlinenum

行先頭に行番号をつけるアプリ <https://caseynelson314.github.io/addlinenum/>

左側のテキストエリアにコードを入力すると、右側に行番号がついたコードが表示されます。シンタックスハイライトはお飾りでコピーされません。

Word 等に張り付けると殺伐としたレポートに彩が。
